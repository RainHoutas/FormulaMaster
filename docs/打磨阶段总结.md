## 四、关键踩坑（防再次踩）

> 详见 [`memory/HANDOFF_TO_NEXT_SESSION.md`](../C:/Users/Houtas/.claude/projects/F--AppProject-FormulaMaster/memory/HANDOFF_TO_NEXT_SESSION.md) "注意事项"段。

精选 Top 10：

1. **KSP 版本固定** `2.3.2`（Kotlin 2.2.10 / AGP 9.2.0）
2. **WebView 主线程**：`WebViewPool.acquire/release` 必须在主线程
3. **WebViewPool release 必须显式 `loadUrl("about:blank")`**：否则 50-300ms 内复用时显示旧 KaTeX 内容
4. **WebView `useWideViewPort` 必须 false**：默认 true 时 viewport 用屏幕设备宽度而非 WebView frame
5. **DataStore 是 cold Flow**：共享数据应在 Repository/Preference 内 `stateIn(applicationScope)` 升级为 hot StateFlow
6. **NavHost 默认 transition 不能全 None**：会让旧页面 dispose 但屏幕缓冲未刷新就被覆盖造成"切换瞬间残留"
7. **`pointerInteropFilter` 不能与父级 scrollable 共存**
8. **`MotionEvent.TOOL_TYPE_PALM`**（API 33+）：硬编码常量值 `5` 最稳妥
9. **JS `body.scrollHeight` 是 CSS px**（在 `useWideViewPort=false` 下 = dp），Kotlin 不要再走 density 二次转换
10. **WebView 内容高度自适应必须用 `ResizeObserver`** 监听 KaTeX 字体异步加载触发的 reflow
