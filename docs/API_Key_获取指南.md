# Formula Master API Key 获取指南

Formula Master 使用**云端 OCR API** 把你的手写笔画转成 LaTeX 公式。
这一步在云端完成，所以你需要自己申请 API Key。本指南覆盖目前支持的两路云端方案。

> **Key 怎么存？** 每个 Key 在设备本地用 AES256-GCM 加密（主密钥放在 Android Keystore 硬件安全模块），**绝不上传到 Formula Master 的服务器**——本应用根本没有服务器。Key 只在你的手机本地，每次调 OCR 时直接发给对应服务商。

## 选哪一路？

> ⚠️ **请先看清楚门槛**：两路 API 都需要前置付费才能开通开发者账号，但金额差异很大。

| 方案 | 国内可用 | 前置门槛 | 后续日常免费额度 | 实际花销（自用学生场景） |
|---|---|---|---|---|
| **A2 SimpleTex** | ✅ 直连 | **必须充值 ≥¥20** 进账户 | Turbo 2000/天 + 标准 500/天 | ≈ ¥20 一次性，几乎一直在免费额度内 |
| **A1 Mathpix Snip** | ⚠️ 需代理 | **$19.99 (≈¥145) 激活费 + 信用卡** | 仅一次性 $29 启动金 ≈ 14500 张图 | $19.99 + 后续按量 $0.002/image |

### 我应该选哪个？

- **国内学生 / 想低成本** → **充 ¥20 配 A2 SimpleTex 即可**，正常使用基本一直在免费额度内，这 ¥20 实际是"放在那不动"的余额池。
- **海外用户 / 愿意为最强准确率付钱** → A1 Mathpix 是行业最强 OCR，配上 A2 兜底也合理。
- **没有国际信用卡** → 只能选 A2。Mathpix 注册环节卡死在信用卡上。

> ### SimpleTex ¥20 充值是什么？
>
> SimpleTex 官方公告（2025 年起）：
> > "近期检测到一些恶意用户批量注册刷取免费 API 调用量，为了防止这一现象发生，现新注册开放平台时需要预先充值 20 元以上的金额进入您的账户，且不支持因错误理解开放平台是什么而进行退款。"
>
> **解读**：
> - 这 ¥20 是**充值到你的账户余额**，不是手续费 —— 你将来超额使用时它会被扣减
> - 正常使用日常都在免费额度内（Turbo 2000/天 + Standard 500/天），¥20 实际"躺"在账户里
> - 但**一旦充进去就别指望退**——官方明示不退款
> - **重要**：如果你只是想用 SimpleTex 网页版或客户端识别公式（不开发应用），**不要来注册开放平台**，直接用 [simpletex.cn](https://simpletex.cn) 主页面是免费的

> ### Mathpix 付费明细
>
> Mathpix 已经不是"个人开发者免费试用"模式了。目前激活 API 必须 $19.99 一次性付费 + 绑定 Visa/Mastercard。$29 启动金可以覆盖约 14500 张图（按 $0.002/image），用完之后按量付费。**对于自用学习量极大的用户，长期使用累计费用并不高**（1000 张图 ≈ $2），但前置门槛比 SimpleTex 高得多（¥145 vs ¥20，且 SimpleTex 的 ¥20 是充值不是消费）。

## A2：申请 SimpleTex UAT Token（推荐路径）

> 📖 **官方文档入口**（如有疑问以官方为准）：
> - API 总览：<https://simpletex.cn/api_doc>
> - 详细文档：<https://doc.simpletex.cn/zh/api/>
> - UAT 鉴权说明：<https://doc.simpletex.cn/zh/api/auth_uat.html>
> - 公式识别 API：<https://doc.simpletex.cn/zh/api/api_formula_recognition.html>

### 1. 注册账号

1. 访问 **<https://simpletex.cn>** 主站，点右上角 **"注册"**
2. 用手机号注册（邮箱注册可能也支持，以官网为准）
3. 完成验证码 / 短信验证后登录

> 国内直连，无需代理。如果偶遇打不开，过几小时再试。

### 2. 进入用户中心

登录后访问 **<https://simpletex.cn/user/center>**，进入用户中心。

### 3. ⚠️ 充值 ¥20（必经步骤）

**SimpleTex 自 2025 年起的反恶意注册政策**：开发者开放平台账号必须先充值 **≥¥20** 才能继续操作（创建 token / 调 API）。

操作流程：
1. 用户中心找 **"充值"** / **"账户余额"** 入口
2. 充值至少 **¥20**（多充也可以，不会用得快）
3. 支持微信 / 支付宝等国内支付方式

**重要提醒**：
- 这 ¥20 是**充值到账户余额**，**不是手续费**——免费额度内的调用不会扣它
- 但官方明确**不支持因"误解开放平台是什么"退款**，所以**别冲动多充**
- 如果你只想用 SimpleTex 公式识别功能（不做开发），**直接用 [simpletex.cn](https://simpletex.cn) 主站 / 客户端 / 网页版**就行，无需开通开放平台

### 4. 生成 UAT Token

1. 在用户中心左侧菜单找 **"用户授权令牌"**
2. 点 **"创建新令牌"** / **"+ 新建"**（按钮名以实际页面为准）
3. 给令牌起个备注名，例如 `Formula Master`
4. 创建成功后会显示一长串 token 字符串

### 5. 复制 Token

**整段一次性复制**（包括所有字母数字符号，不要漏字符）。

> ℹ️ SimpleTex 官方文档 ([auth_uat.html](https://doc.simpletex.cn/zh/api/auth_uat.html)) 提示：
> "为确保您的应用与账户安全，请不要在生产环境中使用本方法进行请求"
> 这是针对**对外服务的服务端**而言（怕你把 token 写进自己的 Web 后端被日志泄露 / 抓包嗅探）。
> Formula Master 是个人手机本地应用，UAT 全程在你自己手机上 + Tink AES256-GCM 加密存储 + HTTPS
> 传输到 SimpleTex 服务器，不存在"我们的服务器代用户调 API"的中转链路，使用 UAT 完全合规。
>
> SimpleTex 也提供更复杂的"**App 授权**"方式（APP_ID + APP_SECRET + MD5 签名），适合你自己搭
> 后端代理的场景。本应用没有后端，所以选了 UAT。详见 [App 授权官方文档](https://doc.simpletex.cn/zh/api/auth_app.html)。

### 6. 免费额度（务必了解）

SimpleTex 公式识别 API 当前免费政策（[官方文档 2025-03-31 版](https://doc.simpletex.cn/zh/api/api_formula_recognition.html)）：

| 端点路径 | 模型 | 每日免费额度 | QPS 限制 | 本应用是否使用 |
|---|---|---|---|---|
| `/api/latex_ocr` | 标准模型，准确率优先 | **500 次/天** | 2 | ✅ 已接入 |
| `/api/latex_ocr_turbo` | 轻量模型，速度优先 | **2000 次/天** | 5 | ✅ 已接入 |

> 完整域名：`https://server.simpletex.cn/api/...`（注意是 **.cn** 不是 .net）

- **配置一次 token，两个模型同时启用**：在设置页填一次 UAT，下拉里会同时出现 "SimpleTex Turbo" 和 "SimpleTex 标准" 两个选项
- **推荐绑定**：Light（实时预览）→ **Turbo**（高频不心疼）/ Deep（精确识别）→ **标准**（关键识别更准）
- **额度独立计算**：两个端点的免费额度互不挤占
- **QPS 限制**：本应用 Light 档防抖间隔 300ms，Deep 档 1.5s，正常使用不会触发 QPS 限流（除非你超快连续抬笔落笔）
- **何时重置**：每日重置（请以官方为准）
- **超额怎么办**：按官方阶梯收费（具体单价见 [SimpleTex 价格](https://simpletex.cn)）；超额前会有提醒
- **是否需要预存**：注册时可能要求 ¥5-20 预存防欺诈，不会扣到免费额度内的请求

> **额度估算**：默认推荐组合（Light=Turbo 2000/天 + Deep=标准 500/天）下，**每天可写 200-400 道公式不会超额**——基本无限自由用。日常学生场景完全足够。

## A1：申请 Mathpix Snip API Key（付费）

> ⚠️ **此节针对海外或愿意付费的用户**。国内用户没有国际信用卡的，请直接跳过本节，只配 A2 即可。

### 0. 先看清楚费用

启用 Mathpix Convert API **必须**满足以下两条：

1. **支付 $19.99 一次性激活费**（账号开通后第一次进入开发者控制台时收取）
2. **绑定信用卡 / 借记卡**（用于后续 pay-as-you-go 扣费）

注册后会自动获得 **$29 启动金**，可用于：
- 约 **14500 张** OCR 图片（$0.002/image）
- 或约 **5800 页** PDF（$0.005/page）

启动金用完后按量自动从绑定的卡扣费。**没有持续免费层**。如果你只是偶尔用，启动金能撑很久；如果是日常密集使用，做好按月几美元的心理预期。

### 1. 注册账号

1. 访问 **<https://accounts.mathpix.com/login>**
2. 点 **"Sign Up"** 注册
3. 用邮箱或 Google 账号注册均可
4. 验证邮箱后登录

> 服务器在境外，国内**直连大概率失败**，建议开稳定代理。

### 2. 进入 Console

访问 **<https://console.mathpix.com>**，登录。

### 3. 完成组织（Organization）设置

第一次进 Console 会引导你完成：

1. 创建 **Organization**（个人用户也需要建一个）
2. 提示支付 **$19.99 setup fee**
3. 添加 **Credit/Debit Card**

完成这三步后你才能创建 API App。

### 4. 创建 App 拿到 App ID + App Key

1. Console 左侧菜单进 **"OCR"** → **"Apps"**（具体路径可能因 UI 改版略有不同）
2. 点 **"Create App"**
3. 给 App 起名，例如 `formula-master-android`
4. 创建成功后页面会显示：

| 字段 | 长度 | 注意 |
|---|---|---|
| **App ID** | 短字符串 | 随时可见 |
| **App Key** | 64 字符长字符串 | **只在创建时完整显示一次**！关掉页面后只能看部分掩码 |

**两个都要复制下来**。如果 App Key 没及时复制，需要删 App 重建。

### 5. 看用量

Console 的 **"Usage"** 页可以看实时调用次数和余额。

## 在 Formula Master 中配置

### 第一次配置

1. 打开 Formula Master，点底栏 **"设置"** Tab（齿轮图标）
2. 找到 **"识别器配置"** 卡片：
   - **A2 SimpleTex**：在 UAT Token 输入框填入 token
   - **A1 Mathpix Snip**（仅当你已付费拿到 Key）：在 App ID 和 App Key 两个输入框分别填入对应值
3. 每张卡片底部点 **「保存」**
4. 点 **「测试连接」** 验证 Key 是否有效
   - ✅ **连接正常**：绿色对勾，可以用了
   - ❌ **Key 无效或已过期**：检查复制是否完整（有时复制少一两位）
   - ❌ **网络超时**：Mathpix 多见，检查代理；SimpleTex 检查网络
   - ❌ **无网络连接**：手机离线了

### 绑定到识别档位

往下滚到 **"识别档位绑定"** 区：

1. **「实时预览（300ms 防抖）」** 下拉：选一个识别器
2. **「精确识别（1.5s 防抖）」** 下拉：选一个识别器

**推荐组合**：

| 用户类型 | Light 推荐 | Deep 推荐 | 说明 |
|---|---|---|---|
| 只配了 SimpleTex（推荐主流路径） | **SimpleTex Turbo** | **SimpleTex 标准** | 同一 token 双绑定，2000+500/天额度互不挤占，**几乎不会超额** |
| 同时配了 Mathpix + SimpleTex | SimpleTex Turbo | Mathpix Snip | Turbo 跑高频预览（2000/天烧得起），Mathpix 留给最终精度把关 |
| 只配了 Mathpix（付费用户） | Mathpix Snip | Mathpix Snip | 全部走付费 API，最强准确率 |

**只配了 SimpleTex token？** 下拉里会有 Turbo 和 Standard 两个选项，建议如上分别绑定到 Light/Deep。

**未绑定状态**：下拉选「未绑定」表示该档不触发识别（写字也不发请求）。两档都未绑定时识别功能整体停摆，预览条和候选区都会保持空。

## 切换 / 撤销 Key

### 在应用内修改

进设置 Tab，找到对应识别器卡片，**直接覆盖输入框 → 保存** 即可。无需重启。

### 撤销已泄露的 Key

如果你怀疑 Key 泄露（推送代码到公开仓库后忘删等）：

- **SimpleTex**：登 <https://simpletex.cn/user/center>，到"用户授权令牌"页面删除当前 token，新建一个
- **Mathpix**：登 console.mathpix.com，删除对应的 App，重新 Create App 拿新的 App ID/Key

旧 Key 服务端撤销后立即失效，应用本地的旧 Key 会在下一次「测试连接」时显示「Key 无效或已过期」。

### 一键重置

设置页底部「**重置所有配置**」按钮会清掉本应用存的所有 Key 和绑定（**不影响**服务端的 Key 本身——那是你账号的，需要在对应 Console 单独删）。

## 常见问题排查

### Q: 没有国际信用卡，能用 Mathpix 吗？

**不能**。Mathpix 注册必须绑定 Visa/Mastercard 等国际卡，国内储蓄卡不接受。请用 A2 SimpleTex。

### Q: SimpleTex 每天的额度够用吗？

按推荐组合（Light=Turbo 2000/天 + Deep=标准 500/天）：
- Light 档每次停笔 300ms 触发一次（每道公式 5-10 次）→ 烧 Turbo 额度
- Deep 档每次停笔 1.5s 触发一次（每道公式约 1 次）→ 烧标准额度
- **每天约 200-400 道公式不会超额**，正常学生每天 30-100 道绰绰有余

如果真的超额，会按官方阶梯收费（很便宜）。也可以暂时把 Light 档设为「未绑定」省 90% 额度。

### Q: 测试连接显示成功，但写公式后预览/候选都是空？

不是 Key 问题，是识别失败：
- 笔画太少：尝试写完整一些
- 字迹潦草：服务端可能识别不出
- 笔画位置太靠边：画布不要写太满，留出边距

### Q: 识别返回的 LaTeX 看起来怪怪的？

我们在识别器输出上接了 `LatexNormalizer` 做后处理（Unicode→LaTeX 命令、函数名补全、花括号补齐等）。如果发现某个公式总是出现奇怪结果，可以反馈截图 + 实际输出 + 期望结果。

### Q: 我用了 VPN 但 Mathpix 还是连不上？

- 检查 VPN 是否覆盖了 Java/原生应用（不是只代理浏览器）
- 检查 Android 系统 → 设置 → 网络和互联网 → VPN 是否在用
- 改用 SimpleTex 也是合理的备选

### Q: Key 输入是密文显示的，怎么确认我没填错？

每个输入框右边有「**显示** / **隐藏**」切换按钮，点一下就能看明文核对。

### Q: 卸载应用会发生什么？

本地存储全部清除，包括加密的 Key。重装后需要重新填 Key（服务端的 Key 仍有效）。
